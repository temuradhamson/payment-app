# Payment Tracker

FastAPI + SQLite приложение для записи платежей.

## Стек
- Python 3.14
- FastAPI
- Jinja2
- SQLite

## Запуск
```bash
python -m venv venv
venv\Scripts\pip install -r requirements.txt
venv\Scripts\uvicorn main:app --reload
```
Или просто запусти `start.bat`

Откройте http://localhost:8000
